Zero Context Cost.
Infinite Compute.

Sub-agents can execute Python scripts in isolated subprocesses. Data processing, computations, ML pipelines — run without consuming LLM context tokens. Pure Python. Pure speed.

01

Isolated Execution

Each Python RPC runs in its own subprocess with temp working directories. No cross-contamination between tasks.

02

Streaming Output

Async generator yields stdout/stderr lines as they arrive. Real-time progress visible in the Web-UI timeline.

03

pip Integration

Auto-detect Python installations. Install packages on-demand into isolated directories. Check availability before execution.

04

JSON Arguments

Pass structured args via PYTHON_RPC_ARGS env var. Access with os.environ.get(). No manual serialization needed.
